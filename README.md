# XOR
La puerta lógica XOR representa una operación fundamental selectiva. A diferencia de la suma lógica (OR), que acepta cualquier combinación donde al menos una entrada sea verdadera, la XOR solo devuelve un estado verdadero cuando sus entradas presentan valores opuestos entre sí. Esto significa que si ambas señales son iguales (ya sean ambas 0 o ambas 1), el resultado será siempre 0; solo cuando una entrada es "1" y la otra es "0" se activa la salida. Esta propiedad de "detección de disparidad" la convierte en esencial para construir circuitos como los sumadores binarios, donde el XOR se encarga de calcular la suma de los bits individuales antes de procesar el acarreo.
